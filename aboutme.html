Reflections

July 19th, 2017

After dabbling with both Python and Scratch, I realized there are many similarities between the two programming languages. I can build loops, conditionals, variables and functions along with making things go forward and turn left or right. However, there are no sprites readily available in Python and the syntax is very different as well. I can't drag and drop any blocks in Python like what I did in Scratch.

My group and I also started debugging our programs and it was pretty frustrating when more errors popped up after "fixing" one part of the code. Although, it was time consuming, I did like challenging myself and thinking critically. Problem solving is a great skill to practice.

July 21st, 2017

In class today, we learned about data structures and how we can create a list in Python. A list is basically an array of data and it is very useful when you need to store a list of coordinates or a list of pixels in a picture for example. It stores a large set of data in one place so it's easy to access the information in one go. After learning about them, we stored all the rgb pixels of one picture in a list, classified each set of pixels in one intensity category from low to high and changed the color of each intensity level to create our own filters, just like Obama's Hope campaign poster.

July 25th, 2017

Today we built a BOE robot and I liked this activity a lot! We learned about circuits and how we can light up a LED using jumper wires, resistors and a battery. Although, my group and I spent an hour figuring out how to light up the LED using a series and parallel circuit, I enjoyed trying out different wire positions to get the circuit to work.

July 27th, 2017

One thing I learned about circuits is that power comes from GND so in order to power the circuit, you need to connect a jumper wire from Ground to the breadboard. I also learned that current flows away from the negative side of the battery to the positive side so the positive lead of a LED should be further away from GND. After testing our circuits, we made our robot dance by using servos and adding in go forward, go backwards, turn left and turn right functions. We made out robot turn 360 degrees and sway from left to right. It was fun watching other group's robots dance as well.

August 1, 2017

After experimenting with Javascript today, I learned how to add a button that changes the background color each time you click on it and an alert box that pops up everytime the page loads. Some of my peers have really cool features as well. Carolina has a menu bar that slides in an out of the page and Brittany has icons and buttons that link her website to projects she made throughout the program. One challenge I faced today was trying to randomize the rgb values so a random background color would appear every time the user clicks the button. I'm still stuck on the problem but I'll keep working on this.
