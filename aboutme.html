Reflections

July 19th, 2017

After dabbling with both Python and Scratch, I realized there are many similarities between the two programming languages. I can build loops, conditionals, variables and functions along with making things go forward and turn left or right. However, there are no sprites readily available in Python and the syntax is very different as well. I can't drag and drop any blocks in Python like what I did in Scratch.

My group and I also started debugging our programs and it was pretty frustrating when more errors popped up after "fixing" one part of the code. Although, it was time consuming, I did like challenging myself and thinking critically. Problem solving is a great skill to practice.

July 21st, 2017

In class today, we learned about data struuctures and how to create a list in Python. A list is basically an array of data and it is very useful when you need to store a list of coordinates or a list of pixels in a picture for example. It stores a large set of data in one place so it's easy to access the information in one go. After learning about them, we used our knowledge and created our own filters.
